## Description

[Bookmark Service Api](https://github.com/ivandi1980/bookmark-service-api) is backend service api using [NestJS](https://nestjs.com/) framework written by `TypeScript`.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
